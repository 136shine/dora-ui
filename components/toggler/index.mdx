# Toggler 切换器

Toggler 是一个用于控制切换（一个布尔值）的 `render props` 组件。

常见使用场景：show/hide 某个组件，使用较为局限，没有 `hooks` 灵活。

## API

| 属性           | 说明                       | 类型      | 默认值                |
| -------------- | -------------------------- | --------- | --------------------- |
| children       | renderProps 所需函数       | function  | -                     |
| defaultToggled | 默认状态值                 | boolean?  | false                 |
| afterToggled   | 当切换行为发生后触发的回调 | function? | (toggled:boolean)=>{} |
