# Toast 轻提示

基于 Popup 组件（position:center），提供 success/error/info/loading 等形态。

- Toast.success(content,duration,callback,mask)
- Toast.error
- Toast.info
- Toast.loading

另外提供一个销毁方法：

- Toast.destroy

注意： Toast.loading 不会自动消失，需要手动调用 Toast.destroy

## 示例

### 只有文字提示的 toast

<Playground>
  <Button
    onClick={() => {
      Toast.info('系统升级中，分享功能暂停使用', 1000);
    }}
  >
    only text
  </Button>
</Playground>

### 持续时间为 3s 的 toast

<Playground>
  <Button
    onClick={() => {
      Toast.success('3s success toast', 3000);
    }}
  >
    toast keep 3s
  </Button>
</Playground>

### 带回调的 toast

<Playground>
  <Button
    onClick={() => {
      Toast.error(
        'error toast',
        2000,
        () => {
          console.log('关闭后的回调');
        },
        false
      );
    }}
  >
    toast with callback
  </Button>
</Playground>

### 带蒙层的 toast

<Playground>
  <Button
    onClick={() => {

  >

  </Button>

### loading

<Playground>
  <Button

## API

| 属性     | 说明             | 类型      | 默认值            |
| -------- | ---------------- | --------- | ----------------- |
| content  | toast 内容       | ReactNode | -                 |
| duration | 持续时间（毫秒） | Number?   | 1500,loading 无效 |
| onClose  | 关闭回调         | Function? | ()=>{}            |
| mask     | 是否展示蒙层     | Boolean?  | false             |
